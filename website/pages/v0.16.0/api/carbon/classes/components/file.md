## class `File`

Each file component can only display 1 attached file, but you can upload multiple files and add them to different file components within your payload.

### Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| type | `unknown` | Yes |  |
| isV2 | `unknown` | Yes |  |
| file | ``attachment://${string}` | undefined` | Yes | The attachment to display in the file component. |
| spoiler | `unknown` | Yes | Whether the file should be displayed as a spoiler. |
| serialize | `unknown` | Yes |  |
