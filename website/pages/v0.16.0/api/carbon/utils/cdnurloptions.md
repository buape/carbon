## interface `CDNUrlOptions`

Options for building CDN URLs

### Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| format | `ImageFormat` | No | The image format to use @default "png" |
| size | `ImageSize` | No | The image size to use (must be a power of 2 between 16 and 4096) |
