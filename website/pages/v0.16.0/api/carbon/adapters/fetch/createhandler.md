## `createHandler(client: Client | ClientManager): Handler`

Creates a fetch handler function for the client or client manager routes

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| client | `Client | ClientManager` | Yes | The client or client manager to create the handler for |

### Returns

`Handler`
