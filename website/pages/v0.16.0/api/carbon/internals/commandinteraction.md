## class `CommandInteraction`

Represents a command interaction

### Properties

| Name | Type | Required | Description |
|------|------|----------|-------------|
| options | `OptionsHandler` | Yes | This is the options of the commands, parsed from the interaction data. It will not have any options in it if the command is not a ChatInput command. |
