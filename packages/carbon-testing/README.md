This package is part of [the Carbon framework](https://github.com/buape/carbon), check it out for more details.

`@buape/carbon-testing` is a test harness for Carbon bots. It attaches to the Carbon `Client` you already export, replaces Discord HTTP calls with a fake Discord transport, and records Carbon-level responses so you can test commands, components, modals, autocomplete, followups, and Discord REST calls without hitting Discord.
